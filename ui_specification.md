## 1. ЭКРАН "ГЛАВНАЯ" (Home Screen)

### Назначение
Отображение личных наборов карточек пользователя, быстрый доступ к изучению.

### Компоненты экрана

#### Header (Шапка)
```
┌─────────────────────────────────────┐
│ ☰  Flashcards           🔍  [+]     │
└─────────────────────────────────────┘
```

**Элементы:**
- **Меню-бургер** (☰) - открывает боковое меню (drawer)
- **Логотип/Название** - "Flashcards" или название приложения
- **Поиск** (🔍) - кнопка поиска по наборам
- **Добавить** ([+]) - создать новый набор

#### Сводная статистика (Daily Summary Card)
```
┌─────────────────────────────────────┐
│  📅 Сегодня                         │
│                                     │
│  🔥 Streak: 7 дней                  │
│  📝 Карточек на изучение: 23        │
│  ✅ Изучено сегодня: 15             │
│                                     │
│  [Начать изучение] ───────────────► │
└─────────────────────────────────────┘
```

**Данные:**
- Текущий streak (дни подряд)
- Количество карточек, ожидающих повторения
- Уже изученных сегодня карточек
- Кнопка быстрого старта изучения всех карточек на сегодня

#### Фильтры и сортировка
```
┌─────────────────────────────────────┐
│ [Все наборы ▼] [По дате ▼] [⚙️]     │
└─────────────────────────────────────┘
```

**Фильтры:**
- Все наборы / Избранное / Недавние / По категориям
- Сортировка: По дате создания / По названию / По прогрессу / По активности

#### Список наборов (Card Sets List)

**Каждый набор отображается как карточка:**

```
┌─────────────────────────────────────┐
│ 🇬🇧 English Vocabulary              │
│ 150 карточек                        │
│                                     │
│ ████████░░ 80% изучено              │
│                                     │
│ 🔴 12 на сегодня                    │
│ 🟡 35 изучаются                     │
│ 🟢 103 изучено                      │
│                                     │
│ Последнее изучение: 2 часа назад    │
│                                     │
│ [Изучать]    [⋮]                    │
└─────────────────────────────────────┘
```

**Элементы карточки набора:**
1. **Иконка категории** - эмодзи или иконка (языки, наука, история и т.д.)
2. **Название набора** - крупным шрифтом
3. **Количество карточек** - общее количество
4. **Прогресс-бар** - визуальный индикатор изученных карточек
5. **Статистика по карточкам:**
   - 🔴 Красный - карточки на сегодня (просроченные + запланированные)
   - 🟡 Желтый - изучаются (в процессе повторения)
   - 🟢 Зеленый - изучено (mastered)
6. **Последняя активность** - когда последний раз учил
7. **Кнопка "Изучать"** - быстрый старт
8. **Меню (⋮)** - дополнительные действия

#### Меню набора (долгое нажатие или ⋮)
```
┌─────────────────────────┐
│ Изучать                 │
│ Редактировать           │
│ Добавить в избранное    │
│ Поделиться              │
│ Экспортировать          │
│ Сбросить прогресс       │
│ Удалить                 │
└─────────────────────────┘
```

#### Пустое состояние (Empty State)
Если у пользователя нет наборов:

```
┌─────────────────────────────────────┐
│                                     │
│          📚                         │
│                                     │
│   Создайте свой первый набор       │
│        карточек!                    │
│                                     │
│  Начните изучать новые темы с       │
│  помощью интервальных повторений    │
│                                     │
│     [Создать набор]                 │
│                                     │
│          или                        │
│                                     │
│  [Импортировать из файла]           │
│                                     │
└─────────────────────────────────────┘
```

#### FAB (Floating Action Button)
```
                                  ┌───┐
                                  │ + │
                                  └───┘
```
- Плавающая кнопка справа внизу
- Основное действие: Создать новый набор
- Цвет: Primary
- Тень для выделения

## 2. ЭКРАН "СОЗДАНИЕ/РЕДАКТИРОВАНИЕ НАБОРА" (Create/Edit Set Screen)

### Назначение
Создание нового набора или редактирование существующего.

### Header
```
┌─────────────────────────────────────┐
│ ← Новый набор              ✓ Готово │
└─────────────────────────────────────┘
```

**Элементы:**
- **Назад (←)** - возврат (с подтверждением если есть изменения)
- **Заголовок** - "Новый набор" или название набора
- **Сохранить (✓)** - сохранить изменения

### Форма создания набора

```
┌─────────────────────────────────────┐
│                                     │
│  📸 [Выбрать иконку]                │
│                                     │
│  ┌───────────────────────────────┐  │
│  │ Название набора *             │  │
│  └───────────────────────────────┘  │
│                                     │
│  ┌───────────────────────────────┐  │
│  │ Описание (опционально)        │  │
│  │                               │  │
│  │                               │  │
│  └───────────────────────────────┘  │
│                                     │
│  Категория                          │
│  ┌───────────────────────────────┐  │
│  │ Выберите категорию... ▼       │  │
│  └───────────────────────────────┘  │
│                                     │
│  Теги (через запятую)               │
│  ┌───────────────────────────────┐  │
│  │ английский, базовый, A1       │  │
│  └───────────────────────────────┘  │
│                                     │
│  ☐ Сделать набор публичным          │
│                                     │
│  ──────────────────────────────────│
│                                     │
│  Настройки изучения                 │
│                                     │
│  Новых карточек в день: [20]  [-][+]│
│  Повторений в день: [100]     [-][+]│
│                                     │
└─────────────────────────────────────┘
```

**Поля формы:**

1. **Иконка набора** (опционально)
   - Выбор из эмодзи
   - Загрузка изображения
   - Выбор цвета фона

2. **Название набора*** (обязательно)
   - Текстовое поле
   - Максимум 100 символов
   - Валидация: минимум 1 символ

3. **Описание** (опционально)
   - Многострочное поле
   - Максимум 500 символов

4. **Категория**
   - Dropdown список
   - Варианты: Языки, Наука, История, Математика, Медицина, Программирование, Другое
   - Можно создать свою категорию

5. **Теги**
   - Текстовое поле
   - Разделитель: запятая
   - Используются для поиска

6. **Публичный доступ**
   - Чекбокс
   - Если включено - набор появится в библиотеке

7. **Настройки изучения**
   - Лимит новых карточек
   - Лимит повторений

### Действия после создания

```
┌─────────────────────────────────────┐
│   ✓ Набор успешно создан!           │
│                                     │
│  [Добавить карточки]                │
│  [Импортировать карточки]           │
│  [Вернуться на главную]             │
└─────────────────────────────────────┘
```

**Компоненты формы:**

   - Кнопка добавления изображения
   - Кнопка добавления аудио
   - Превью добавленных медиа
   - Счетчик символов (0/5000)

   - Аналогично лицевой стороне

   - Подсказка (hint) - отображается при изучении
   - Теги - для категоризации

### Панель инструментов форматирования

```

└─────────────────────────────────────┘

- **B** - жирный
- **I** - курсив
- **U** - подчеркивание
- **🎨** - цвет текста/выделение
- **⎘** - вставить код/формулу
- **@** - вставить специальный символ
